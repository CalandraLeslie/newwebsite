React Cookbook Application

Project information

  • Category: React
  • Course: React/Web framework
  • Project date: Spring 2024
  • Project URL: https://github.com/CalandraLeslie/cookbook

This project was an application using React and Vite to create a search function displaying meals from an API.
This was one of the most challenging assignments I have had so far in my studies.
Each meal after you do a search, displays the name and nationality of the dish and it has a hover function.
I found this assignment challenging but it taught me to be patient and I learned a lot from it!
